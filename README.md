# 🏈 NFL 2024 · What It Takes To Win
**Power BI Dashboard · Josué Arcia**

## About this project
Analysis of all 32 NFL teams in the 2024 season to identify 
what separates playoff teams from eliminated ones.

## Key Findings
- 8 of the last 10 Super Bowl winners had a Top-5 defense
- Playoff teams scored 30% more points per game (26.4 vs 20.2 PPG)
- The NFC North was the strongest division with 11.2 avg wins
